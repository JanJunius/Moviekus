Moviekus ist ein Übungsprojekt mit dem Schwerpunkt Xamarin-Forms unter Visual-Studio. Es beinhaltet eine Android- und ein UWP-App unter Anwendung des MVVC-Patterns, 
Einbindung des Entity-Frameworks und noch einiger weiterer Bibliotheken.

Inhaltlich geht es um die Verwaltung von Filmen, die auch über eine REST-API aus MovieDb übernommen werden können.

Neben der App enthält die Solution auch noch ein Web-Projekt. Auch dieses dient reinen Übungszwecken zum Erlernen von Razor-Pages und generell 
der Umsetzung mittels ASP.NET.
